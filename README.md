## How to get the data files

1. Download all the files here: https://drive.google.com/drive/folders/1Uf-LebLWyi9eCv97iBal7kL1NgiGEsv_

2. Make a new folder /data

3. Put them all in there

4. Done!

## Task 1 — malware category models

1. Install dependencies: `pip install -r requirements.txt`
2. Open and run `task1_malware_category_classification.ipynb` from the project root (folder that contains `data/`).
3. For final reported results, keep `QUICK_MODE = False` in the first code cell. A full run can take a long time; `QUICK_MODE = True` uses fixed hyperparameters to verify the notebook runs end-to-end.
4. Export tables and metrics from the notebook into your PDF report (team roles, algorithms, preprocessing, train/test split, hyperparameter method, accuracy + per-class TPR/FPR, comparison).
